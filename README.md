# Note-Taking App
## Developed with:
- MERN full-stack.
## Features
- User login, signup, and logout.
- Auto-login when page refreshes and auto-logout after jwt expires.
- CRUD operations.
- Share notes with guest users setting expiration time and read_only or read_and_write permissions.
